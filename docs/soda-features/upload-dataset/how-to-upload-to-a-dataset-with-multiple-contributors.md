# Overview

The Upload Dataset feature supports multiple contributors uploading to the same dataset. Read the page further to learn how to upload when the contributors are using different computers or the same computer.

## Uploading to the same dataset with multiple contributors - Different Computers

If you are one of many contributors to a dataset but are not sharing the same computer you can simply follow the steps outlined [here](./step-3.md) to upload to a dataset with multiple contributors.

## Uploading to the same dataset with multiple contributors - Same Computer

If you are one of many contributors to a dataset and are sharing the same computer you can follow the following steps:

1. Navigate to Upload a SDS compliant dataset
2. Navigate to step 3. If the dataset already exists on Pennsieve choose 'Upload to an existing dataset' and select the dataset you want to upload to. If the dataset does not exist on Pennsieve, choose 'Upload to a new dataset' and create a new dataset.
3. Upload your dataset. Once the upload is complete click the 'Save & Continue' button. This will remove your progress file and allow another user to create a separate progress file for the same dataset.

:::info
For guidance on uploading to a dataset that already exists on Pennsieve, please refer to the [Upload to an existing dataset](./step-3#uploading-data-to-an-existing-dataset-on-pennsieve) section.
:::
