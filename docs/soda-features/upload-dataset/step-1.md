Sign in if you have not already signed in yet. Sign in to your Pennsieve account and choose the workspace where you want to upload your dataset.

![](/img/upload-dataset-pennsieve-logged-out.png)

After signing in, confirm the selected account and workspace are correct.

![](/img/upload-dataset-confirm-account-details.png)

Make sure the Pennsieve Agent is installed before continuing. If it isn't installed, SODA will prompt you to install it. If you encounter any problems, see the [Installing the Pennsieve agent](../../miscellaneous/common-errors/installing-the-pennsieve-agent) guide.
