## Background

The SODA app saves all errors in several log files on your computer. In order to solve user issues, the SODA Team usually needs to take a look at those log files.

## How to gather the log files

To automatically gather your log files:

1. Navigate to the SODA landing page (the page you see when you first open the SODA app):
   - If you are using the Prepare Dataset Step-by-Step mode, click `Save and Exit` to navigate to the landing page.
   - If you are in the Upload Dataset mode:
     - If you have not started an upload, click the `Contact Us` page in the sidebar.
     - If you have started an upload, click the `Exit` button to return to the landing page.
2. Select the `Contact Us` tab in the sidebar.
3. Click `Gather Logs` and then select a location to store the log files.
4. They will be stored in a zip file named `SODA-logs.zip` in the location you selected.

## How to send the log files to the SODA team

Once you have the log files you can send them to us using the SODA contact form available (here)[/contact-us].
