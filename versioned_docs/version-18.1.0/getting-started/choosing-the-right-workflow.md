Not sure which workflow to use? Read the criteria listed under each option below and select the one that applies to your dataset.

## Prepare a Dataset Step-by-Step

**Use this if:**

- You are submitting a dataset to Pennsieve for the first time
- Your dataset is **not yet** SPARC Dataset Structure (SDS) compliant
- You need help organizing and preparing your data

This workflow guides you through preparing your dataset to meet SDS requirements and then provides the ability to create a local copy of the standardized dataset as well as upload it to Pennsieve.

## Upload an SDS-Compliant Dataset

**Use this if either of the following is true:**

- Your dataset is already SPARC Dataset Structure (SDS) compliant and was prepared outside of SODA
- You are uploading data to an existing dataset on Pennsieve (as a collaborator or with additionally collected data)

This workflow lets you upload an SDS-compliant dataset directly to Pennsieve or add data to an existing dataset.
